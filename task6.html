GIT flow

GIT clone - для того, чтобы "склонить" репозитор;

GIT pull - для того, чтобы получить изменения с репозитором;

GIT checkout-b branchname - для того,создать новую ветку и переключиться на неё;

GIT add - добавить изменения, которые сделали локально;

GIT commit-m "commit text" - для того, чтобы подписать их;

GIT push origin branchname - запушить на git сервер;
